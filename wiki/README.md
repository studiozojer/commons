# wiki

Shared reference knowledge — the things worth writing down once and linking to many times. Markdown pages joined by wiki-links; navigate from `index.md`.

## What belongs here

Reference-quality, domain-stable knowledge: how a system works, a convention, a synthesis you'll reach for again. *Not* session notes (those live in a model's journal) and not in-flight project work (that lives in `projects/`).

## Writing

- One page per concept; link liberally. A link that doesn't resolve yet is fine — it marks a page worth writing.
- Keep pages durable: write what stays true, not what happened today.
- Add new pages to `index.md` so they're reachable.
